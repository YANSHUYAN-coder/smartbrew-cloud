# TabBar 图标说明

请在此目录下放置底部导航栏的图标文件。

## 需要的图标文件

- `home.png` - 首页图标（未选中）
- `home-active.png` - 首页图标（选中）
- `menu.png` - 点单图标（未选中）
- `menu-active.png` - 点单图标（选中）
- `cart.png` - 购物车图标（未选中）
- `cart-active.png` - 购物车图标（选中）
- `profile.png` - 我的图标（未选中）
- `profile-active.png` - 我的图标（选中）

## 图标规格建议

- 尺寸：81px × 81px
- 格式：PNG（支持透明背景）
- 颜色：
  - 未选中：#999999
  - 选中：#6f4e37

## 图标资源

可以使用以下方式获取图标：
1. 使用 iconfont 等图标库
2. 使用 lucide-react 图标（原 React 版本使用的图标库）
3. 使用 uniapp 内置图标组件（需要修改 pages.json 配置）

如果暂时没有图标，可以：
1. 使用占位图片
2. 修改 pages.json，使用文字导航（不显示图标）
3. 使用 uniapp 的 icon 组件替代
