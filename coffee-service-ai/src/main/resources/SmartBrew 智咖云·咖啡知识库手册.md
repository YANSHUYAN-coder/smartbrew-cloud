SmartBrew 智咖云·咖啡知识库手册

第一章：我们的咖啡豆

1.1 核心拼配豆：SmartBlend No.1 (意式拼配)

产地：哥伦比亚（胡伊拉）50% + 巴西（喜拉多）30% + 云南（保山）20%。

烘焙度：中深烘焙 (Medium-Dark Roast)。

风味描述：入口具有明显的黑巧克力与烤坚果风味，尾韵带有焦糖的甜感，油脂（Crema）丰厚，非常适合制作拿铁和卡布奇诺。

咖啡因含量：中等。

1.2 单品豆：埃塞俄比亚·耶加雪菲 (Yirgacheffe)

产地：埃塞俄比亚耶加雪菲产区，海拔 1800-2000米。

处理法：水洗处理 (Washed)。

烘焙度：浅中烘焙 (Light-Medium Roast)。

风味描述：经典的茉莉花香，带有明亮的柠檬柑橘酸质，口感干净如茶。建议制作手冲或冷萃，不建议加奶。

1.3 单品豆：云南·花魁 (Yunnan Huakui)

产地：中国云南普洱。

处理法：厌氧日晒 (Anaerobic Natural)。

风味描述：浓郁的热带水果香气（百香果、菠萝），伴有发酵酒香。

第二章：经典饮品制作标准

2.1 美式咖啡 (Americano)

配方：双份浓缩 (Double Espresso, 36g液重) + 250ml 热水/冰水。

口感：纯粹、解腻、提神。

推荐：适合喜欢黑咖啡、追求低热量的顾客。

2.2 拿铁 (Latte)

配方：双份浓缩 + 280ml 蒸奶 (奶泡厚度 0.5cm)。

口感：奶香浓郁，咖啡味柔和。

区别：相比卡布奇诺，拿铁的奶泡更薄，牛奶比例更高。

2.3 澳白 (Flat White)

配方：双份精粹浓缩 (Ristretto, 25g液重) + 200ml 蒸奶 (极薄奶泡)。

口感：相比拿铁，咖啡味更浓郁，口感更丝滑。

第三章：冲煮建议 (给购买咖啡豆的顾客)

3.1 手冲参数建议 (V60滤杯)

粉水比：1:15 (例如 15g 粉注入 225g 水)。

水温：浅烘焙豆建议 90-93℃；深烘焙豆建议 85-88℃。

研磨度：中细研磨（类似白砂糖颗粒）。

注水手法：三段式注水（闷蒸30g -> 注水至120g -> 注水至225g）。

3.2 摩卡壶建议

研磨度：细研磨（比手冲细，比意式粗）。

注意：下壶建议加热水，以缩短萃取时间，避免焦苦味。
